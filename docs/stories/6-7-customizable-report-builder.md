# Story 6.7: Customizable Report Builder

**Story ID:** 6.7
**Epic:** 6 (Visualization, Reporting & Export)
**Status:** Done

## User Story

As an analyst,
I want to select exactly which charts and tables go into my report,
So that I can tailor the document to the specific audience or investigation focus.

## Acceptance Criteria

### AC 1: Report Builder UI
- [x] A drag-and-drop or checkbox UI allowing selection of available widgets (Protocol Dist, Top Talkers, Geo Map, Timeline, etc.).
- [x] Option to add custom section headers or text blocks.

### AC 2: Preview
- [x] Live preview of the report layout before generation.

## Design & Implementation

### Component Structure
- **`ReportBuilder.tsx`**: Main UI for composition.
