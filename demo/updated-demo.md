# NetPack Parser Demo Recording Walkthrough

## Summary
Demo video recordings showcasing all major features of the NetPack Parser application across 8 epics and 105 Functional Requirements.

---

## Demo Recordings

### 1. UI Tab Exploration Demo
![UI Exploration Demo](/Users/delphijc/Projects/net-pack-parser/demo/demo_ui_exploration.webp)

**Features Demonstrated:**
- Overview tab with dashboard components
- Packet Inspector for packet analysis
- Timeline for chronological visualization
- YARA Rules management
- Threat Intel database
- IOC Manager for indicators of compromise
- False Positives tracking
- Settings page with import/export
- Remote Capture configuration
- Live Capture interface
- Performance monitoring page

---

### 2. Performance & Live Capture Demo
![Performance & Capture Demo](/Users/delphijc/Projects/net-pack-parser/demo/demo_performance_capture.webp)

**Features Demonstrated:**
- Core Web Vitals monitoring (LCP, FCP, CLS, FID, TTFB)
- Performance score gauge (0-100)
- Server health monitor
- Network interface selection
- Live capture controls (Start/Stop)
- Remote capture authentication
